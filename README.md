# 🕋 Al-Noor (আল-নূর) - Islamic App

আল-নূর (Al-Noor) একটি আধুনিক ইসলামিক ওয়েব অ্যাপ্লিকেশন, যা ব্যবহারকারীদের দৈনন্দিন ইবাদত পালনে সাহায্য করার জন্য তৈরি করা হয়েছে। এটি নামাজের সঠিক সময়সূচী, পবিত্র কুরআন এবং হাদিস পড়ার সুবিধা প্রদান করে।

## ✨ প্রধান বৈশিষ্ট্যসমূহ (Key Features):
* **নামাজের সময়সূচী (Prayer Times):** বাংলাদেশের বিভিন্ন জেলা অনুযায়ী নামাজের সঠিক সময়।
* **পরবর্তী নামাজের কাউন্টডাউন:** পরবর্তী নামাজের জন্য কত সময় বাকি আছে তা লাইভ দেখা যায়।
* **৩০ পারা কুরআন:** পবিত্র কুরআনের সহজ পাঠ।
* **হাদিস ভাণ্ডার:** প্রতিদিনের জন্য সহীহ হাদিসের সমাহার।
* **ডিজিটাল তসবীহ:** জিকির ও তাসবিহ গণনার সহজ সমাধান।
* **আরবি হরফ শিক্ষা:** নতুনদের জন্য আরবি হরফ ও উচ্চারণ শিক্ষা।

## 🛠️ প্রযুক্তি (Tech Stack):
* **Frontend:** HTML5, Tailwind CSS
* **JavaScript:** Vanilla JS
* **API:** Aladhan Prayer Times API & Alquran Cloud API
